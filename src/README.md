## ğŸš€ Getting Started

### **Prerequisites**
- Node.js 18+ (for development)
- Modern web browser (Chrome, Firefox, Safari, Edge)
- npm or yarn package manager

### **Installation**

```bash
# Clone the repository
git clone https://github.com/yourusername/origin-tiles.git

# Navigate to project directory
cd origin-tiles

# Install dependencies
npm install

# Start development server
npm run dev
```

### **Environment Variables**

Create a `.env` file in the root directory:

```env
# Google Maps API (for Dealers Locator)
VITE_GOOGLE_MAPS_API_KEY=your_google_maps_api_key_here

# Optional: Analytics
VITE_GA_TRACKING_ID=your_ga_id_here
```

**Note:** The site includes safe environment variable checks. Maps gracefully degrade if API key is missing. All other features work without environment variables.

### **Build for Production**

```bash
# Create optimized production build
npm run build

# Preview production build locally
npm run preview
```

- **Vercel** (recommended) - Zero config deployment

- **AWS S3 + CloudFront** - Enterprise scale
